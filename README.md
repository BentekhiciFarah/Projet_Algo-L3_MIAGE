Le but était de réaliser des fonctions demandées par le prof dans le cadre d'un projet du module "Algorithmique" en langage C, afin de mettre en pratique les notions aqcuises pendant le semestre. 
Les fonctions tournaient principalement autour des structures de données et l'optimisation de la complexité du code : 
  - Les arbres.
  - Les listes chainées.
  - Les tableaux.
  - Les piles et les files.
